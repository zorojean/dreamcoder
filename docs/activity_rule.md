一、赛事背景
DreamField 是一个提供 AI 算力服务的平台，面向开发者、创作者、团队及企业提供 AI 模型调用与算力使用能力。随着 AI 产品、Agent、工作流工具和 AI 辅助创作项目快速增长，优秀作品需要更持续的展示渠道、真实用户反馈和算力资源支持。

DreamSeed 种梦计划 — AI创造者大赛由 DreamField 发起，旨在通过线上作品比赛的形式，降低 AI 创作与应用验证的算力门槛，让更多普通用户、开发者和创作者能够以更低成本使用 AI 算力，尝试产品原型、工具、Agent 和 AI 辅助创作。

本赛事希望为创作者提供作品推广、榜单曝光、用户反馈和现金奖励机会，帮助优秀作品从“完成创作”走向“被看见、被使用、被持续迭代”；同时通过助梦者与种子机制，让更多用户参与 AI 作品成长过程，共同推动 AI 应用生态和 AI 产业发展。

三、赛事角色
3.1 造梦者
报名提交 AI 相关作品并通过审核的用户称为“造梦者”。造梦者需完成 DreamField 账号注册、GitHub 账号绑定与认证，并提交符合赛事要求的公开 GitHub 仓库。

3.2 助梦者
未提交作品或希望支持作品的用户，可以绑定一个有效参赛作品成为“助梦者”。助梦者使用 DreamField 的 AI 算力调用可生产种子，种子用于提升绑定作品在星耀榜中的排名与曝光。

七、参赛作品要求
本赛事面向 AI 产品、skill、工具、Agent、AI 辅助创作项目或其他 AI 相关作品。单个 DreamField 账号可以报名多个仓库，仓库独立参与排名、晋级和奖金评定；造梦者本人每日折扣只按其名下有效参赛仓库中的最高档发放。

报名仓库需同时满足以下要求：

仓库为公开 GitHub 仓库。
仓库创建时间距报名提交时间不超过 30 天。
仓库属于当前绑定的 GitHub 账号。
仓库内容与 AI 相关，或明确体现由 AI 辅助创作完成。
仓库使用 DreamField 官方 README.md 模板，且模板必须放在 README.md 顶部。
参赛仓库需永久关联 DreamField 官方 README.md 模板，并持续保留 README.md 中的 DreamField 活动编号、活动名称、参赛声明、活动链接和其他 DreamField 相关信息，不得删除、更改或弱化相关内容。
仓库不得包含恶意代码、泄露的密钥、未授权个人信息、明显侵权内容、违规定向采集内容或其他违反法律法规及平台规则的内容。
用户中途解绑 GitHub 账号的，其报名资格、排名资格、奖金资格及未使用权益视为作废。

八、报名审核
报名提交后，系统优先进行自动审核。自动审核范围包括 DreamField 账号状态、GitHub 绑定状态、仓库归属、仓库公开状态、仓库创建时间、官方 README.md 顶部模板、README.md 中 DreamField 相关信息保留状态、重复报名和基础内容安全检查。

作品状态包括：

待审核：报名已提交，系统或官方人员尚未完成审核。
已上架：审核通过，作品可参与榜单展示、排名计算、折扣发放和助梦绑定。
已下架：审核未通过或后续复核不符合规则，作品不参与榜单展示、排名计算、折扣发放和助梦绑定。
待复核：作品存在信息变更、用户申诉、系统异常或其他需官方确认的情况。
自动审核未通过时，作品进入已下架状态，系统应向造梦者展示不通过原因。造梦者可以根据原因修改仓库或报名信息后重新提交，DreamField 官方人员也可进行二次审核。二次审核结果以 DreamField 官方记录为准。

九、榜单与排名规则
9.1 种梦榜
种梦榜是本次赛事主榜，展示有效参赛作品按 GitHub 总 Star 数形成的排名。现金奖励、阶段结果和每日折扣均以种梦榜及 DreamField 官方复核结果为准。

每日 00:00，DreamField 读取有效参赛仓库的 GitHub Star 数据并锁定当日榜单。若多个作品 Star 数相同，可并列显示名次；涉及阶段名额、现金奖励或权益边界时，DreamField 可结合活动记录进行复核，并以官方公示或通知结果为准。

种子投放不直接改变种梦榜排名、阶段结果或现金奖励名次。

9.2 星耀榜
星耀榜展示参赛作品收到的种子支持情况，用于提升作品曝光，帮助更多用户发现正在被支持的作品。星耀榜不直接决定现金奖励名次。

9.3 蹬王榜
蹬王榜展示用户持有和参与种子互动的情况，仅用于活动展示和运营互动，不影响种梦榜、现金奖励和折扣档。

十一、种子获得与用途
助梦者绑定一个有效参赛作品后，使用 DreamField 的 AI 算力调用即可生产种子。每个用户同一时刻只能绑定一个作品；未绑定有效参赛作品时，不产生种子。造梦者也可以作为助梦者，但只能绑定自己名下一个已参赛作品。

种子会投放给用户当前绑定的作品，用于提升作品在星耀榜中的排名与曝光，助力造梦者作品被更多用户看见。种子也会计入用户的蹬王榜展示，但不直接改变种梦榜排名、阶段结果或现金奖励名次。

种子的发放、投放、可用状态、提回、扣除和展示以 DreamField 系统记录为准。种子不是现金、积分或可提现资产，不可要求 DreamField 折现、兑付现金或开具现金奖励证明。

十二、绑定、解绑与重绑
每个用户同一时刻只能绑定一个作品。绑定操作全天可进行，解绑操作仅支持每日 09:00 至 18:00。

助梦者解绑时，可将其投给原作品的可提回种子提回账户。DreamField 收取 5% 手续费，手续费部分作废，剩余种子向下取整入账。

重新绑定后，账户现有种子会一次性投给新绑定作品。新折扣从次日 00:00 起按新绑定作品的当日种梦榜折扣档生效。

团队获奖的，应指定唯一奖金接收主体。团队内部奖金分配、成员授权、成员争议、二次分配税务问题由团队自行处理，与 DreamField 无关。
